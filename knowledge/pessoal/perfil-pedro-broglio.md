- Trata agentes como membros da equipe — delega, cobra, espera autonomia
- Kobe (OpenClaw) é o agente operacional: Telegram, crons, integrações
- Claude Code é o agente de desenvolvimento: código, deploy, pesquisa
- Espera que agentes consultem o [[memory/context/business-context|contexto]] antes de agir
- Se o agente erra, quer entender o porquê e corrigir o sistema, não só o sintoma
- Salva decisões e contexto no [[PROPAGATION|segundo cérebro]] para persistência entre sessões
