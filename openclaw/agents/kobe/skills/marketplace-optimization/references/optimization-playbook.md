## Sequência de implementação {#sequencia}

### Mês 1 — Fundação (sem custo extra, impacto alto)

**Semana 1-2:**
- [ ] Auditar todos os títulos — ML (60 chars), Shopee (120 chars), Amazon (150 chars)
- [ ] Preencher fichas técnicas ML a 100% (material, dimensões, capacidade, cor, marca)
- [ ] Ativar 20 tags no Shopee para cada produto
- [ ] Responder 100% das perguntas abertas nas 3 plataformas
- [ ] Verificar rating/reputação nas 3 plataformas e identificar causas de queda

**Semana 3-4:**
- [ ] Verificar elegibilidade Full ML para potes herméticos de vidro
- [ ] Criar variações de produto onde não existem (tamanhos + quantidades)
- [ ] Ativar Product Ads ML com campanha automática (budget: R$30-50/dia por top SKU)
- [ ] Configurar respostas automáticas no chat Shopee
- [ ] Auditar fotos — mínimo 6 no ML, 9 na Shopee, 7 na Amazon

### Mês 2 — Aceleração

- [ ] Entrar no catálogo ML com produto mais vendido (preço + Full + reputação verde)
- [ ] Ativar Full ML nos top 3 SKUs (se elegível)
- [ ] Participar de Flash Sale Shopee (inscrever produtos 1-2 semanas antes)
- [ ] Ativar Shopee Ads (Search Ads) nas 5-10 keywords principais — campanha manual
- [ ] Gravar e publicar vídeo nos listings Shopee (smartphone + boa iluminação é suficiente)
- [ ] Otimizar campanhas ML com base em dados do mês 1 (migrar keywords boas para manual)
- [ ] Criar listings Amazon com estrutura otimizada (título + bullets + backend keywords)

### Mês 3 — Expansão

- [ ] Criar kits exclusivos por plataforma (composições diferentes para cada canal)
- [ ] Solicitar Brand Registry Amazon usando registro INPI Budamix
- [ ] Criar A+ Content Amazon para top 3 SKUs
- [ ] Planejar e executar primeira Shopee Live (demonstração de organização de despensa)
- [ ] Ativar Amazon Sponsored Products para top SKUs (auto campaign primeiro)
- [ ] Calcular viabilidade FBA por SKU — considerar peso do vidro
- [ ] Analisar dados de keyword ML Ads e ajustar lances

### Mês 4-6 — Consolidação

- [ ] Construir estratégia de reviews nas 3 plataformas
- [ ] Otimizar precificação cruzada (ML base, Shopee -5-10%, Amazon +5%)
- [ ] Criar conta Instagram @budamix para tráfego externo → Amazon
- [ ] Shopee Live recorrente (1-2x por mês para construir base de seguidores)
- [ ] Monitorar BSR Amazon e ajustar PPC conforme ranking
- [ ] Ativar FBA se viabilidade confirmar para os SKUs principais
- [ ] Analisar concorrentes nos top keywords — identificar gaps de otimização

## Quick Wins detalhados {#quick-wins}

### 1. Títulos com keyword no início
**Esforço:** 1-2h | **Impacto:** Alto | **Custo:** Zero

Atualizar todos os títulos para ter keyword principal nos primeiros chars:
- ML: `Pote Hermético Vidro [X]ml [atributo] Budamix [conjunto/kit]`
- Shopee: `Budamix Pote Hermético Vidro [X]ml Tampa Bambu [uso]`
- Amazon: `Budamix Pote Hermético de Vidro [X]ml Tampa Bambu | Kit [N] Potes [uso]`

### 2. Fichas técnicas ML a 100%
**Esforço:** 2-3h | **Impacto:** Alto | **Custo:** Zero

Campos críticos para potes herméticos:
- Material (vidro borossilicato)
- Capacidade (ml/L)
- Dimensões (altura × diâmetro)
- Cor (transparente, tampa bambu)
- Marca (Budamix)
- Modelo/coleção
- Compatibilidade (micro-ondas: sim/não, freezer: sim/não, lava-louças: sim/não)
- País de origem

### 3. Tags Shopee (20 tags por produto)
**Esforço:** 30min | **Impacto:** Alto | **Custo:** Zero

Template de tags para potes herméticos:
```
pote vidro, pote hermético, organizador mantimentos, vasilha vidro, kit potes,
pote com tampa, frasco vidro, organização despensa, pote vidro hermetico,
conjunto potes vidro, pote tampa bambu, vidro cozinha, pote guardar alimentos,
pote café, pote açúcar, organização cozinha, budamix, pote hermético vidro,
pote hermetico, kit organização
```

### 4. Participar de Flash Sale Shopee
**Esforço:** 30min de inscrição | **Impacto:** Muito alto | **Custo:** Desconto no produto

- Acessar Seller Centre → Central de Marketing → Flash Sale
- Inscrever pelo menos 7-10 dias antes
- Selecionar produto com melhor histórico de vendas
- Definir preço de flash sale (mínimo 5% de desconto sobre preço normal)
- Resultado: spike de vendas → impulsiona ranking orgânico nas semanas seguintes

### 5. Vídeo nos listings Shopee
**Esforço:** 2-3h (gravação + edição simples) | **Impacto:** Alto | **Custo:** Zero

Roteiro para vídeo de pote hermético (45 segundos):
```
0-5s: Close no produto na embalagem / apresentação da marca Budamix
5-15s: Mostrar produto de perto — vidro transparente, tampa bambu, qualidade
15-25s: Demonstrar hermeticidade — fechar, virar de cabeça pra baixo com líquido
25-35s: Mostrar uso — organizando despensa, colocando café/açúcar/farinha
35-45s: Kit completo com os tamanhos / call to action "Adicione ao carrinho"
```

Ferramentas: CapCut (gratuito) para edição simples. Fundo branco ou despensa organizada.

### 6. Responder perguntas abertas
**Esforço:** 1h | **Impacto:** Direto em conversão | **Custo:** Zero

Perguntas mais comuns para potes herméticos de vidro:
- "Vai no micro-ondas?" → Depende do modelo; especificar na resposta e incluir na descrição
- "Qual a capacidade exata em litros?" → Informar ml e L
- "A tampa sela bem?" → Descrever mecanismo de vedação
- "Vai na lava-louças?" → Informar compatibilidade
- "É de vidro mesmo ou plástico?" → Confirmar material e tipo de vidro
- "Vem com quantas unidades?" → Especificar se kit ou unitário

**Incluir essas respostas NA DESCRIÇÃO para reduzir abandono.**

## Long Game — estratégias de médio prazo {#long-game}

### Construção de Reviews (2-4 meses)
**ML:**
- Reviews têm peso moderado — processo natural
- Qualidade do produto e embalagem são os melhores "gatilhos" de review

**Shopee:**
- Configurar mensagem automática 3-5 dias após entrega
- Usar "Moedas por Avaliação" — usuário ganha moedas ao deixar review com foto/vídeo
- Inserir papel dentro da caixa com QR code para o listing Shopee + pedido de avaliação

**Amazon:**
- Solicitar Review via Seller Central (botão "Request a Review" por pedido)
- Não pedir review diretamente por email — policy proíbe
- Para novos SKUs: solicitar Amazon Vine Program
- Responder todos os reviews negativos com solução visível

### MercadoLíder Platinum (3-6 meses)
- Requisito atual: 1.725+ vendas / R$296.000+ em 60 dias
- Com pico histórico de R$1M/mês, o status Platinum é o target natural
- Verificar se a queda de faturamento tirou o status — e quais métricas estão bloqueando

### Star Seller Shopee (2-3 meses)
- Focar em: chat 100% respondido em < 12h, envio no prazo, zero pontos de penalidade
- Verificar critérios exatos no Seller Centre de Shopee
- Badge de Star Seller = boost visual + melhor posicionamento

### Brand Registry Amazon + A+ Content (1-2 meses)
- Budamix já tem registro INPI → elegível para Brand Registry Amazon
- Processo: Amazon Brand Registry (brandregistry.amazon.com.br) → registrar marca
- Benefícios: A+ Content, Sponsored Brands, Brand Store, proteção anti-counterfeit

### Shopee Live Recorrente (3-6 meses para construir base)
- Frequência ideal: 2x por mês
- Formato: demonstração prática + oferta exclusiva para espectadores
- Conteúdo para potes herméticos: "Como organizar sua despensa em 30 min"
- Meta de 3 meses: base de 1.000+ seguidores da loja
- Longo prazo: lives geram tráfego recorrente sem custo de ADS

### Tráfego Externo para Amazon (2-4 meses para ver resultado)
- Criar @budamix no Instagram e TikTok focado em organização de casa
- Conteúdo: antes/depois de despensas organizadas com potes Budamix
- Link na bio: direto para Store da Amazon
- Parcerias com home organizers / influencers de organização (microinfluencers são mais custo-efetivos)
- Meta: 500-1000 cliques externos/mês para Amazon → impacto real no A10

- [ ] Categoria correta (não anunciar em categoria errada)
- [ ] Título: 60 chars, keyword principal no início
- [ ] 6-7 fotos (fundo branco + detalhes + lifestyle + infográfico + dimensões)
- [ ] Ficha técnica: 100% dos campos
- [ ] Descrição: responde as 6 dúvidas principais
- [ ] Preço: competitivo vs. top 5 da categoria
- [ ] Frete: Full ML (se elegível) ou Coleta

- [ ] Variações: criar se houver mais de 1 tamanho/quantidade

- [ ] Título: 120 chars, keyword principal nos primeiros 30 chars

- [ ] Descrição: keywords naturais + resposta às dúvidas principais

- [ ] Frete Grátis / Frete Patrocinado ativado
- [ ] Variações criadas se houver tamanhos/quantidades diferentes

### Checklist de Lançamento — Novo SKU na Amazon
- [ ] Brand Registry ativo
- [ ] Parent-child configurado (produto pai + variações)
- [ ] Título: até 150 chars com keyword no início
- [ ] 5 bullets otimizados com keywords em maiúsculo no início

## Framework de diagnóstico {#diagnostico}

### "Por que meu anúncio caiu de posição?"

**Passo 1 — Verificar métricas básicas:**
```
ML: Reputação ainda verde? → Se não, corrigir cancelamentos/atrasos/reclamações
Shopee: Pontos de penalidade? → Se > 0, identificar causa e corrigir
Amazon: ODR > 1%? → Risco de suspensão, prioridade máxima
```

**Passo 2 — Verificar estoque:**
```
Produto ficou sem estoque? → Causa mais comum de queda
Amazon: Ficou fora por > 3 dias? → Esperar 2-4 semanas de recuperação com PPC agressivo
ML/Shopee: Ficou fora? → Reativar e aumentar velocidade de vendas via promoção
```

**Passo 3 — Verificar concorrência:**
```
ML: Concorrente entrou no catálogo? → Verificar preço, Full e condições
Shopee: Concorrente com mais reviews? → Acelerar programa de avaliações
Amazon: BSR de concorrente subiu? → Comparar listing — keywords, fotos, preço
```

**Passo 4 — Verificar qualidade do listing:**
```
Título foi alterado recentemente? → Mudança de título pode afetar indexação por 1-2 semanas
Fotos foram atualizadas? → Nova foto principal que não converte pode derrubar CTR
Preço foi aumentado? → Verificar impacto em conversão
```

### "Por que minhas conversões caíram?"

**Checar na sequência:**
1. Foto principal ainda boa? (CTR impacta conversão)
2. Preço saiu da faixa competitiva?
3. Reviews negativos recentes puxaram a nota para baixo?
4. Dúvidas críticas sem resposta no listing?
5. Concorrente com oferta/desconto especial?

## Precificação e margem {#precificacao}

### Estrutura de custo por plataforma (aproximado):

| Custo | ML | Shopee | Amazon |
|-------|----|----|--------|
| Comissão plataforma | 11-17% | 12-16% | 11-15% |
| Frete (Full/FBA) | Variável | Variável | Variável |
| ADS (% do faturamento) | 5-15% | 5-12% | 8-20% |
| Embalagem e operação | 3-5% | 3-5% | 3-5% |
| **Total de custo canal** | **~20-30%** | **~20-28%** | **~22-35%** |

**Regra de bolso:** Produto precisa ter margem bruta ≥ 40% para suportar custos de canal + ADS e ainda ter lucro.

### Estratégia de precificação para potes herméticos Budamix:

**Referência:** Preço ML = base (100%)
- Shopee: 95% do preço ML (mais agressivo para competir com genéricos)
- Amazon: 105% do preço ML (público premium, disposto a pagar por Prime)

**Kits como proteção de margem:**
- Kit 3 potes = menor custo por unidade de ADS + menor custo de frete por unidade
- Ticket médio maior = mais fácil atingir limites de frete grátis
- Percepção de valor agregado = resistência a comparação de preço

**Sazonalidade:**
- Janeiro (organização pós-réveillon): alta demanda por organizadores
- Outubro-novembro (Black Friday): aumentar estoque e preparar campanhas
- Meses de campanha Shopee (8.8, 9.9, 10.10, 11.11, 12.12): preparar estoque e ADS
