# Lições — RH

_Atualizado: 2026-05-15_

## 2026-05-15
- O compliance diário não deve depender apenas do relatório de saída; para cobrir CLT, é necessário validar intervalos de almoço e jornada total diretamente nos registros do dia.
- Follow-ups com Regra 18 podem ficar deliberadamente curtos quando o funcionário ainda não engajou; o risco deve ser registrado internamente sem aumentar pressão por WhatsApp fora do fluxo.
- Funcionários com jornada especial, especialmente Leonardo e Mateus, exigem interpretação de contexto antes de classificar atraso, hora extra ou pendência.
