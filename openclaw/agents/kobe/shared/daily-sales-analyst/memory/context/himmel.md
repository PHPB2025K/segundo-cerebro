# Himmel — Contexto ADS para Daily Sales Analyst

## O que é Himmel

Himmel é a gestão de ADS para Shopee e Mercado Livre. Cobre verba, segmentação, criativos, lances, exposição e — no caso da Shopee — afiliados e cupons coordenados.

## Relação com o DSA

- Himmel é **input** para o DSA, não output.
- O DSA recebe contexto ADS/Himmel como parte do pacote de dados.
- O DSA **não fala com Himmel**. Não aciona, não solicita, não recomenda diretamente.
- Quando a decisão tática envolve ADS:
  - Shopee: "Lucas alinha com Himmel"
  - Mercado Livre: "Yasmin alinha com Himmel"

## O que o DSA pode fazer com contexto Himmel

- Usar informação de ADS para contextualizar leitura estratégica/tática.
- Identificar se ADS está sustentando ou mascarando resultado.
- Notar se cupons/afiliados estão inflando métricas.
- Sugerir alinhamento via responsável do canal.

## O que o DSA NÃO pode fazer

- Acionar Himmel diretamente.
- Recomendar mudança de verba/lance/segmentação diretamente ao Himmel.
- Tratar Himmel como responsável primário de ação.
