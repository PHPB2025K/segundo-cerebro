## Fonte

- Canal: WhatsApp próprio do Kobe via Evolution API.
- Grupo: **💎 [HML*] BUDAMIX**.
- Escopo: Mercado Livre / planejamento operacional da conta.
- Status de acesso: Kobe consegue ler mensagens novas legíveis a partir de **2026-05-15 13:11 BRT**.
- Histórico anterior: WhatsApp entregou apenas pacote de histórico sem texto materializado na base Evolution; não considerar como disponível até nova extração viável.

## Regras de uso

- Usar como **contexto operacional e hipótese causal**, não como dado de performance por si só.
- Cruzar sempre com dados reais: vendas, estoque, campanhas, exposição, ADS, preço, ranking e margem.
- Não culpar agência/Himmel sem evidência quantitativa.
- Registrar decisões, planos, projeções, cobranças, responsáveis e janelas de validação.
- Não colar conversa inteira quando não for necessário; resumir o que muda decisão/análise.
