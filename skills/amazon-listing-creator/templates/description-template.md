# Template de Descricao Amazon BR — Budamix

> HTML basico permitido: <br>, <b>, <ul>, <li>
> Sem limite rigido de chars (recomendado 1000-2000)
> Keywords naturalmente distribuidas

## Template

```html
[Nome do Produto] | Budamix<br><br>

[Linha/Design] - [Conceito em 1 frase]<br><br>

[Frase de abertura: beneficio principal]<br><br>

<b>O QUE VEM NO KIT:</b><br>
- [item 1: quantidade + material + dimensao]<br>
- [item 2: quantidade + material + dimensao]<br>
- Total: [N] pecas<br><br>

<b>DESIGN/DIFERENCIAL:</b><br>
[Descricao do diferencial exclusivo em 2-3 linhas]<br><br>

<b>FABRICACAO NACIONAL:</b><br>
[Processo de fabricacao, acabamento, origem]<br><br>

<b>DIMENSOES:</b><br>
[Dimensoes de cada componente]<br>
Peso: [peso]g<br><br>

<b>BUDAMIX</b> - Marca brasileira de Pedreira-SP especializada em utilidades domesticas de design com tecnologia de corte a laser.
```

## Regras
1. HTML permitido: <br>, <b>, <ul>, <li>, <p>
2. NAO usar: <script>, <style>, <img>, <a href>, CSS inline
3. NAO incluir precos, promocoes, links externos
4. NAO mencionar outras plataformas
5. Keywords distribuidas naturalmente
6. Manter objetivo ~1500 chars
