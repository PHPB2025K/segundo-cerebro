## Imagem Única

| Especificação | Valor |
|---|---|
| **Formatos** | JPG, PNG |
| **Proporção** | 1.91:1 (horizontal) a 4:5 (vertical) |
| **Recomendado** | 1:1 (quadrado) — melhor para feed mobile |
| **Resolução ideal** | 1080×1080px (1:1) ou 1080×1350px (4:5) |
| **Resolução mínima** | 600px na menor dimensão |
| **Tamanho máximo** | 30MB |
| **Texto na imagem** | Manter <20% — impacta distribuição mesmo sem limite técnico |
| **Primary text** | 50–150 caracteres (80 char ideal) |
| **Headline** | 27 caracteres |
| **Descrição** | 18 caracteres |
| **URL Display** | Opcional, 30 caracteres |

**Boas práticas:**
- Fundo claro para produtos em fundo neutro (converte melhor)
- Produto ocupando 60–70% da imagem
- Cores da marca + contraste alto para chamar atenção no feed
- Mostrar produto em uso (context shots) > produto em fundo branco
- Incluir preço ou benefício principal quando relevante

## Vídeo Único

| Especificação | Valor |
|---|---|
| **Formatos** | MP4, MOV, GIF |
| **Codec vídeo** | H.264 |
| **Codec áudio** | AAC 128kbps |
| **Frame rate** | Fixo (não variável) — 24fps, 30fps ou 60fps |
| **Proporção Feed** | 1:1 (mais seguro) ou 4:5 (mobile-first) |
| **Proporção Stories/Reels** | 9:16 (obrigatório para nativo) |
| **Resolução mínima** | 1080×1080px |
| **Tamanho máximo** | 4GB |
| **Duração Feed** | 1s a 241 minutos (ideal: 15–30s) |
| **Duração Stories** | 1s a 60s (ideal: 6–15s) |
| **Duração Reels** | 1s a 60s (ideal: 7–15s) |
| **Thumbnail** | JPG/PNG, mesma proporção do vídeo |
| **Legenda** | Fortemente recomendada (65–85% assistem sem som) |

**Estrutura de vídeo (framework hook)**
- **0–3s:** Hook visual — parar o scroll
- **3–8s:** Problema ou situação identificável
- **8–20s:** Solução + produto
- **20–30s:** Prova social + CTA

**Para potes herméticos Budamix:**
- **Hook:** "Toda vez que abro minha geladeira..." ou mostrar desorganização
- **Meio:** Organização com potes, transformação visual
- **CTA:** "Organize sua cozinha hoje →"

## Carrossel

| Especificação | Valor |
|---|---|
| **Cards** | 2 a 10 |
| **Formatos** | JPG, PNG, MP4, GIF (pode misturar) |
| **Proporção** | 1:1 (recomendado) ou 4:5 |
| **Resolução** | Mínimo 1080×1080px |
| **Tamanho imagem** | 30MB por card |
| **Tamanho vídeo** | 4GB por card |
| **Primary text** | 80 caracteres |
| **Headline por card** | 45 caracteres |
| **Descrição por card** | 18 caracteres |
| **Link por card** | Cada card pode ter link diferente |

**Usos estratégicos do carrossel:**
1. **Linha de produtos:** Cada card = 1 produto (com link direto)
2. **Kit/Combo:** Mostrar itens inclusos individualmente
3. **Benefícios sequenciais:** Card 1=problema, 2=solução, 3=prova, 4=CTA
4. **Depoimentos:** Cada card = 1 depoimento de cliente

## Stories e Reels

| Especificação | Valor |
|---|---|
| **Proporção** | 9:16 (obrigatório) |
| **Resolução** | 1080×1920px |
| **Zona segura** | 250px topo e 250px base (nada de texto/CTA aqui) |
| **Duração ideal** | 6–15s |
| **Formatos** | JPG, PNG, MP4, GIF |
| **Texto na imagem** | Manter na zona segura (área central) |

**Para Reels nativos (orgânico + boost):**
- Conteúdo nativo (filmado na vertical, estilo TikTok)
- Não parecer anúncio — maior engajamento
- Primeiros 1–2 segundos determinam o watch rate

## Dynamic Creative Optimization (DCO)

Permite que o Meta teste múltiplas variações automaticamente.

**Limites por criativo DCO:**
- Até 10 imagens/vídeos
- Até 5 primary texts
- Até 5 headlines
- Até 5 descrições
- Até 5 CTAs

**Quando usar DCO:**
- Fase inicial: não sabe o que vai performar
- Budget limitado: Meta concentra nos melhores combos
- **Limitação:** Não dá para ver performance individual de cada combinação

**Quando usar Ads separados:**
- Já testou e tem winners — quer dados granulares
- Quer escalar o criativo específico
- Branding exige consistência do criativo

## Advantage+ Creative

Meta pode automaticamente modificar seus criativos:
- Ajustar brilho/contraste
- Adicionar etiquetas de produto
- Testar variações de texto
- Adaptar proporção por placement

**Para desativar opções específicas** (via API ou Ads Manager):
```json
{
  "creative": {
    "creative_id": "{CREATIVE_ID}",
    "asset_feed_spec": {
      "optimization_type": "REGULAR"
    }
  }
}
```

> Se branding é crítico (Budamix ainda consolidando identidade), considere desativar modificações automáticas inicialmente.

## Checklist de Qualidade do Criativo

**Antes de publicar:**
- [ ] Imagem/vídeo está na proporção correta?
- [ ] Resolução mínima respeitada?
- [ ] Texto ocupa <20% da imagem?
- [ ] Primary text está no tamanho recomendado?
- [ ] Headline tem no máximo 27 chars?
- [ ] Vídeo tem legenda?
- [ ] CTA é claro e específico?
- [ ] URL de destino está correta?
- [ ] Pixel está disparando no destino?
- [ ] Criativo revisado no preview mobile?

**Teste rápido de copy:**
- Leu em 3 segundos? Tem que parar o scroll.
- Tem benefício concreto (não vago)?
- CTA é inequívoco?
