## Lições / guardrails
- Não usar `supabase db push` quando histórico remoto de migrations está desalinhado; aplicar migration específica com backup/smoke.
- Preview Vercel CLI precisa envs públicas explícitas ou Preview Environment configurado.
- Secrets N8N nunca em Code nodes; usar credentials/env.
- App de produção sem git é risco crítico: antes de refactor sério, criar snapshot, baseline git, remote privado e tag de rollback.
