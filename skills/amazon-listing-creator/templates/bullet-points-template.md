# Template de Bullet Points Amazon BR — Budamix

> 5 bullets obrigatorios. Cada um ate 500 chars.
> Primeira palavra em CAPS = keyword principal.
> Sem HTML. Sem precos. Sem promocoes.

## Estrutura Padrao

**Bullet 1 — COMPOSICAO/KIT**
```
KIT COMPLETO COM [N] PECAS - [lista do que vem: quantidade + material + dimensao de cada item]
```
Objetivo: informar exatamente o que o comprador recebe.

**Bullet 2 — DIFERENCIAL EXCLUSIVO**
```
[DIFERENCIAL] EXCLUSIVO - [descricao do que torna o produto unico]. [Beneficio pratico que o diferencial traz]
```
Objetivo: o que so Budamix tem.

**Bullet 3 — FABRICACAO/QUALIDADE**
```
FABRICACAO NACIONAL - [processo de fabricacao]. [Acabamento]. [Material] [espessura/densidade], [durabilidade]
```
Objetivo: justificar qualidade e preco.

**Bullet 4 — USO/APLICACAO**
```
[VERBO DE ACAO] - [problema que resolve]. [Superficies/cenarios compativeis]. Ideal para [lista de usos]
```
Objetivo: mostrar versatilidade e necessidade.

**Bullet 5 — EXTRAS/GARANTIA**
```
[ACESSORIO] INCLUSO - [descricao do extra]. [Dimensoes]. [Uso como presente]. Garantia [periodo]
```
Objetivo: valor agregado e confianca.

## Regras de SEO nos Bullets
- Keywords relevantes nos primeiros 80 chars de cada bullet
- Incluir sinonimos: "porta-copos" + "descanso de copo" + "bolacha"
- Incluir material e dimensoes (clientes filtram por isso)
- Mencionar "Budamix" em pelo menos 1 bullet
- Nao repetir as mesmas keywords entre bullets
