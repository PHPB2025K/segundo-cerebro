### Aprendizados

- **`/tmp` é volátil em macOS** — depois do reboot, o repo Klap em `/tmp/klap-migration/klap-ecommerce/` perdeu até o `.git/objects`. Solução: `gh repo clone PHPB2025K/klap-ecommerce` no mesmo path; tudo que estava em remote sobrevive. Lição: nunca tratar `/tmp` como persistente entre sessões.
- **Amazon SP-API serve a mesma foto com IDs distintos** — não dá pra confiar só em base_id quando a foto pode ter sido catalogada com IDs separados (large vs medium). pHash com PIL+imagehash é a forma confiável de pegar essas duplicatas residuais.
- **`textutil` extrai só texto do `.docx`** — para acessar mídia embutida, usar `unzip -j docx word/media/*` (docx é zip).
