## Sessão noturna — Pipeline DSA ML maduro

Sessão longa de maturação do pipeline Daily Sales Analyst do Mercado Livre. Múltiplas decisões estruturais.

### O que foi feito

- **Verificação do probe ML Ads** — todas as 5 amostras (04:00-06:00 BRT) mostraram ADS já consolidado pro dia anterior; bug do 22/05 foi anomalia pontual da API ML. Conclusão: cron pode rodar cedo (cron oficial agora `50 9 * * *` UTC = 06:50 BRT real).
- **Descoberta de timezone** — VPS Hostinger está em UTC, não BRT. Crontab `0 7 * * *` rodava às 04:00 BRT, não 07:00 BRT como esperado. Memória `timezone_brt_pedreira.md` criada como regra permanente.
- **Sistema MercadoLíder completo** — migration Supabase + fetcher enriquecido + backfill 91 dias + Lente 6 em L01 e L05. Pipeline agora monitora trajetória pra Platinum em todo cron (Budamix: Gold, 80% do caminho).
- **Memory ingest diário** — `daily-memory-ingest-ml.py` integrado como passo [6/6] do wrapper. Fecha ciclo de memória: L05 → weekly.md → input da L01-L03 do dia seguinte.
- **Consolidações semanal + mensal** — prompts + scripts + crons agendados (segunda 09:00 BRT e dia 1 09:00 BRT). Guarda "dados insuficientes" funcionando. Primeira execução real semanal: ~02/06.
- **Benchmark quádruplo de modelos LLM** rodado para o mesmo dia 22/05 com weekly.md populado: Sonnet 4.6 puro, Opus 4.7 puro, GPT 5.5 (via codex), híbrido Sonnet+Opus(L04/L05).
- **Runner refatorado** com `_resolve_model_for_layer` — suporta override de modelo por camada via config (`llm_model_per_layer`). Funciona tanto com `claude -p` quanto `codex exec`.
- **Decisão final consolidada**: modo híbrido no cron diário (Sonnet default + Opus 4.7 em L04/L05) + Opus 4.7 puro nas consolidações. Custo total ML: ~R$ 19,40/mês.

### Decisões

→ Pipeline DSA ML — modo híbrido LLM como default do cron diário. (ver `decisoes/2026-05.md`)
→ Consolidações semanal e mensal usam Opus 4.7 puro.
→ GPT 5.5 (codex CLI) descartado pro pipeline — análise mais rasa, hedging excessivo, bug de formatação.
→ Cron oficial ML movido pra `50 9 * * *` UTC (06:50 BRT real). 5 entradas de probe removidas.
→ Modelos LLM são escolhidos por camada, não por pipeline inteiro.
