# Template de Descricao Shopee — Budamix

> Limite: 3.000 chars. Objetivo: ~1.200-1.500 chars.
> Formato: texto puro. Sem HTML, sem markdown, sem acentos problematicos.
> Usar ◾ para topicos e ▪ para sub-itens.

## Template

```
[Nome do Produto] | Budamix
[Linha/Design] - [Conceito em 1 frase curta]

[Frase de abertura: 1 linha com beneficio principal do produto]

◾ O QUE VEM NO KIT
  ▪ [item 1: quantidade + material + dimensao]
  ▪ [item 2: quantidade + material + dimensao]
  ▪ Total: [N] pecas

◾ DIFERENCIAIS
  ▪ [diferencial exclusivo Budamix]
  ▪ [vantagem sobre concorrentes]
  ▪ [caracteristica tecnica relevante]

◾ FABRICACAO
  ▪ [metodo de fabricacao principal]
  ▪ [acabamento especial]
  ▪ Fabricacao propria em Pedreira-SP
  ▪ Controle de qualidade individual

◾ IDEAL PARA
  ▪ [cenario de uso 1]
  ▪ [cenario de uso 2]
  ▪ [cenario de uso 3]
  ▪ Presente funcional e decorativo

◾ DIMENSOES
  ▪ [dimensao produto principal]
  ▪ [dimensao acessorio se houver]
  ▪ Peso: [peso]g

◾ GARANTIA
  ▪ 90 dias contra defeito de fabricacao

◾ BUDAMIX
Marca brasileira de Pedreira-SP especializada em utilidades domesticas de design com tecnologia de corte a laser.
```

## Regras

1. NUNCA ultrapassar 3.000 chars
2. NUNCA usar HTML ou markdown (<b>, **, ##, etc)
3. NUNCA incluir precos, promocoes ou condicoes de pagamento
4. NUNCA incluir informacoes de contato (telefone, email, WhatsApp)
5. NUNCA mencionar outras plataformas (ML, Amazon, etc)
6. SEMPRE mencionar "Budamix" e "Pedreira-SP"
7. SEMPRE incluir dimensoes em cm/mm
8. SEMPRE incluir peso em gramas
9. Usar linguagem sem acentos para evitar problemas de encoding na API
10. Manter objetivo ~1.500 chars (ninguem le descricao longa na Shopee)
