# Checklist DRE Profissional

## Fórmulas

- Lucro bruto = receita comercial válida - CMV
- Margem bruta = lucro bruto / receita comercial válida
- Resultado operacional conhecido = lucro bruto - taxas marketplace - Ads - gestão/agência - outras despesas operacionais conhecidas
- Margem operacional = resultado operacional conhecido / receita comercial válida
- Resultado líquido gerencial = resultado operacional conhecido - despesas abaixo do operacional disponíveis
- Margem líquida gerencial = resultado líquido gerencial / receita comercial válida

## Red flags

- Settlement usado como faturamento sem qualificador.
- CMV ausente sem marcar DRE como pré-CMV.
- CMV consolidado rateado por marketplace sem nota.
- Shopee Ads via Wallet tratado como performance oficial.
- Margem líquida apresentada como contábil final sem impostos/overhead.
- Arquivo Excel sem validação de formatos numéricos.
- Aba de fontes ilegível ou com texto cortado.
