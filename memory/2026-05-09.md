## Daily Sales Report — correções e decisão de fonte
- Pedro apontou divergência entre o Daily Sales Report de 08/05/2026 enviado no tópico Marketplaces e o valor exibido no Live Sales: report inicial trouxe R$ 9.259,44, enquanto Live Sales mostrava R$ 14.008,87.
- Investigação concluiu que o report inicial usou coletas diretas por API no momento da execução (ML/Shopee/Amazon/Bling), enquanto o Live Sales usa a base canônica do Budamix Central/Supabase (`v_daily_sales`).
- Diferença principal: Amazon foi subcapturada pela coleta direta (R$ 335,30 / 7 pedidos) contra a base canônica (R$ 4.506,76 / 33 pedidos). Shopee também divergiu (R$ 5.224,13 report vs R$ 6.087,03 canônico) e ML teve diferença inversa (R$ 3.700,01 report vs R$ 3.415,08 canônico).
- Decisão operacional: Daily Sales Report deve usar `v_daily_sales` / Live Sales como fonte canônica para marketplaces; APIs diretas ficam como fallback/reconciliação/alerta, não como total oficial do dia anterior.
- Report corrigido de 08/05/2026: total R$ 14.008,87; 248 pedidos; ticket médio R$ 56,49; melhor canal Shopee R$ 6.087,03. Por canal: ML R$ 3.415,08 / 77 pedidos; Shopee R$ 6.087,03 / 138 pedidos; Amazon R$ 4.506,76 / 33 pedidos; Atacado - GB Matriz R$ 0,00.
- Assinatura/rodapé padrão do Daily Sales Report: `_Gerado por Trader | Período: 00:00–23:59 BRT de DD/MM/YYYY_` em itálico, sem bullet, sem prefixo “Trader —”.
