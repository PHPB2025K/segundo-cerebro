# Template de Descricao Shopee — Budamix

> Limite: 3.000 chars. Objetivo: 500-800 chars.
> Formato: texto puro. Sem HTML, sem markdown.
> Usar → para topicos principais e ▪ para sub-itens de especificacoes.
> Estrutura: sussinta, focada em conversao. Cortar tudo que nao ajuda o comprador a decidir.

## Template

```
[Nome do Produto] [Material] [Quantidade] [Diferencial] | Budamix

[1-2 frases: o que e o produto + diferencial principal. Direto ao ponto.]

→ [BENEFICIO PRINCIPAL / USO]
[Explicacao pratica de como o produto funciona ou se adapta. 1-3 linhas no maximo.]

→ ESPECIFICACOES
  ▪ Material: [material + espessura se aplicavel]
  ▪ [Dimensao individual]: [valor]
  ▪ [Dimensao montado se aplicavel]: [valor]
  ▪ Peso: [peso]
  ▪ [Caracteristica tecnica relevante]

→ IDEAL PARA
[Lista corrida em 1 linha, separada por virgulas. Sem bullets individuais.]

→ GARANTIA
90 dias contra defeito de fabricacao.

Budamix — Utilidades domesticas que combinam com voce. Fabricacao propria em Pedreira-SP.
```

## Regras

1. NUNCA ultrapassar 3.000 chars
2. NUNCA usar HTML ou markdown (<b>, **, ##, etc)
3. NUNCA incluir precos, promocoes ou condicoes de pagamento
4. NUNCA incluir informacoes de contato (telefone, email, WhatsApp)
5. NUNCA mencionar outras plataformas (ML, Amazon, etc)
6. NUNCA repetir informacao que ja aparece em outra secao
7. SEMPRE mencionar "Budamix" e "Pedreira-SP" no fechamento
8. SEMPRE incluir dimensoes em cm/mm
9. SEMPRE incluir peso em gramas
10. Usar → para titulos de secao e ▪ para sub-itens de specs
11. Manter objetivo 500-800 chars — sussinto, escaneavel, focado em conversao
12. Secao "IDEAL PARA" em lista corrida (1 linha), nao em bullets separados
13. Eliminar secoes redundantes (ex: "O QUE VEM" + "DIFERENCIAIS" + "FABRICACAO" separados = verboso demais)
